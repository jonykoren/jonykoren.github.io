CT Skull Fractures

A skull fracture is a break in one or more of the eight bones that form the cranial portion of the skull, usually occurring as a result of blunt force trauma.
In this project i'm using deep convolutional networks to improve fractures prediction in skull CT scans.

BMI Image Classifier

Body mass index (BMI) is a value derived from the mass (weight) and height of a person. The BMI is defined as the body mass divided by the square of the body height.
The aim of this repository is to build deep NN classifier that distinguishes between different BMI types of people.

Object Detection

OObject detection is the task of simultaneously classifying and localizing object instances in an image, a video or a real-time.

Web Intelligence

A web crawler, spider, or search engine bot downloads and indexes content from all over the Internet. The goal of such a bot is to learn what (almost) every webpage on the web is about, so that the information can be retrieved when it's needed.
This project presents how to crawl web pages, indexing them within a local search engine and derive the required information easily and quick.
